## Requirements

* verdaccio@3.x or higher

```
 npm install --global verdaccio-audit
```

## Usage
To enable it you need to add this to your configuration file.
```
middlewares:
  audit:
    enabled: true
```

This plugin is experimental and unstable. Please report any issue you found.
